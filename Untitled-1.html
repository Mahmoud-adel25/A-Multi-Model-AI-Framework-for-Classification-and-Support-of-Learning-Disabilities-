A Multi-Model AI Framework for Classification and Support of Learning Disabilities

An AI-powered framework for learning disabilities screening and personalized educational support using deep learning models.

📄 View Final Report 🎥 Watch Demo Video

About the Project

This graduation project presents a multi-model artificial intelligence framework for the classification and support of learning disabilities. Multiple deep learning models are evaluated to detect dyslexia from handwriting while providing educational support through an intelligent application designed for students, teachers, and parents.

Project Features

✍ Handwriting Analysis

Analyze handwriting samples using deep learning.

🧠 Dyslexia Classification

Identify dyslexia using multiple AI models.

📚 Educational Support

Provide AI-powered learning recommendations.

👨‍🏫 Teacher Dashboard

Monitor student performance and progress.

📈 Progress Tracking

Track improvements over time.

Technology Stack
